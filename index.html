Aplikasi Rekon CRM Hitachi

Rekonsiliasi File JRN/TXT untuk Transaksi ATM

Petunjuk Penggunaan

  1. Upload file .jrn atau .txt dari CRM Hitachi (bisa multiple files)
  2. Aplikasi akan mencari keyword: Host Store: Failed, Host Store: Stored, Cash Taken
  3. Untuk setiap keyword yang ditemukan, aplikasi akan mencari nilai AMOUNT terdekat di atasnya
  4. Contoh format amount: AMOUNT : RP 300.000 atau AMOUNT : RP 300,000
  5. Hasil rekonsiliasi akan menampilkan jumlah dan total nilai untuk setiap kategori
